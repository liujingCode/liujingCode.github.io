2.3 gitbook的安装和使用

gitbook的安装和使用

gitbook官网：
1.安装：
    npm install gitbook-cli -g
2.创建一本书
    进入指定文件夹执行 gitbook init
    会在目录下生成SUMMARY.md和README.md两个文件
    SUMMARY.md文件为目录文件
3.在 SUMMARY.md 同级目录下新建配置文件，命名为:book.json
4.加载book.json的配置信息
    gitbook install
5.生成本地网页
    gitbook serve

* [简介](README.md)
